# SellSoft

SellSoft is a single page web application designed to help users easily sell their unused software licenses.
## Tech Stack

This project is built using the following technologies:

- **Frontend**: React.js
- **Styling**: Tailwind CSS
- **Build Tool**: Vite
- **Icons**: React Icons
- **State Management**: React Hooks (useState, useEffect)
- **Deployment**: Vercel

## Features

- **Dark/Light Mode Toggle**: Allows users to switch between dark and light themes for a personalized experience. The toggle button is located in the header and can be activated by clicking.
- **Fully Responsive Design**: The application is optimized for mobile and desktop devices, ensuring a seamless user experience across different screen sizes. It adapts to various devices, such as smartphones, tablets, and desktops, providing a consistent and user-friendly interface.
